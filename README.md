# Внутренний портал

Это веб-сайт для внешнего размещения с двумя разделами: для клиентов и для сотрудников.
Все материалы представлены в открытом виде без авторизации.

## Структура проекта

- `index.html` - Главная страница с выбором категории пользователя
- `styles.css` - Файл стилей для всего сайта
- `/clients/` - Раздел для клиентов
  - `index.html` - Страница с документами для клиентов
  - `/docs/` - Директория с PDF-файлами для клиентов
- `/employees/` - Раздел для сотрудников
  - `index.html` - Страница с документами для сотрудников
  - `/docs/` - Директория с PDF-файлами для сотрудников

## Безопасность

Сайт не содержит авторизации, но реализует следующие меры безопасности:

1. Отсутствие серверного кода - только статические файлы
2. Чистый HTML/CSS без потенциально опасных элементов
3. Все файлы находятся в пределах структуры проекта
4. Нет возможности загрузки файлов пользователями

## Использование

Чтобы использовать сайт:

1. Разместите его на любом хостинге статических файлов
2. Загрузите PDF-файлы в соответствующие директории
3. Обновите ссылки в HTML-файлах при необходимости

Для просмотра локально можно использовать простой HTTP-сервер, например:
```
python3 -m http.server 8000
```
