# Портал инструкций

Современный веб-сайт для внешнего размещения инструкций и документов, разделенных на две категории: для клиентов и для сотрудников.

## Особенности

- Современный, адаптивный дизайн
- Использование SVG-иконок
- Google Fonts (Inter)
- Адаптивность для мобильных устройств
- Безопасный статический сайт без авторизации
- Чистый HTML/CSS/JavaScript

## Структура проекта

```
├── index.html              # Главная страница
├── styles.css              # Стили сайта
├── clients/                # Раздел для клиентов
│   ├── index.html          # Страница для клиентов
│   └── docs/               # Директория с PDF-документами
│       ├── client_instruction.pdf
│       ├── service_guide.pdf
│       ├── terms_of_service.pdf
│       └── frequently_asked_questions.pdf
├── employees/              # Раздел для сотрудников
│   ├── index.html          # Страница для сотрудников
│   └── docs/               # Директория с PDF-документами
│       ├── employee_instruction.pdf
│       ├── work_policy.pdf
│       ├── safety_procedures.pdf
│       └── hr_guidelines.pdf
└── README.md               # Документация
```

## Использование

1. Загрузите все файлы на любой хостинг статических файлов
2. Замените тестовые PDF-файлы в директориях `clients/docs/` и `employees/docs/` на свои документы
3. При необходимости измените ссылки и названия документов в HTML-файлах

## Совместимость

- Все современные браузеры
- Мобильные устройства
- Планшеты

## Безопасность

- Сайт не содержит динамических элементов
- Не требует авторизации
- Все документы доступны в открытом виде
- Подходит для внутреннего и внешнего использования
